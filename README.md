# Telegram Info Bot 🤖

Многофункциональный Telegram-бот на Python. Поддерживает:

- 💬 `/quote` — цитата дня
- 💵 `/usd` — курс доллара
- 🌤 `/weather <город>` — текущая погода
- 🗞 `/news` — главные новости
- 🛠 `/help`, `/info` — описание и справка

## 🚀 Технологии
- Python 3.x
- [pyTelegramBotAPI](https://github.com/eternnoir/pyTelegramBotAPI)
- requests
- dotenv
- OpenWeather API
- News API
- ZenQuotes API
- CBR JSON API
- Flask
